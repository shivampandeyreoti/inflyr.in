🔄 Real-Time Sync (Same Browser)

If you open inflyr.in (or your GitHub Pages site) in another tab on the same browser, this admin panel auto-syncs data via BroadcastChannel + localStorage. Changes made here instantly reflect on the website tab — no server needed.

📤 Export Data

📥 Import Data

ℹ How Data Works

• Where data lives: All data is in localStorage in the browser. It persists until you clear browser data.

• Who sees form submissions: When a creator submits the form on your website, the data goes to localStorage (Pending queue) AND to your email at inflyrhq@gmail.com via FormSubmit.

• Admin panel on GitHub: Since this admin.html is hosted on GitHub Pages (same domain as index.html), they share the same localStorage — no extra setup needed.

• Different devices: Use the Export / Import buttons to move data between devices or browsers.

• Privacy: Creator emails and WhatsApp numbers are stored in localStorage but never shown on the public website. Only admin panel shows them.
